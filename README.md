<h2>🛠️ Installation Steps:</h2>

<p>1. Python 3.12</p>

```
pip install python latest
```

<p>2. Langchain</p>

```
pip install langchain
```

<p>3. Conda Environment</p>

```
conda venv ""
```

<p>4. run</p>

```
streamlit run "filename".py
```
